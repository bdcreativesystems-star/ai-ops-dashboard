# AI Operations Dashboard

A web-based automation dashboard built with Flask that allows users to run operational automations, monitor activity, and generate system reports.

This project demonstrates automation engineering, backend development, logging systems, and dashboard UI design.

---

## Features

- File organization automation
- System reporting tool
- Automation logging
- Flask dashboard interface
- Modular automation architecture

---

## Tech Stack

- Python
- Flask
- HTML/CSS
- Logging system
- Git
